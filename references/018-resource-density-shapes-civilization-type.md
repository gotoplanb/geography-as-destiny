# Resource Density and Gradient Shape Determine Civilization Type

**Source:** Alpha discussion, 2026-04-13
**Type:** connection
**Relevant to:** Chapter 1 (The Distribution), Chapter 2 (The River Spine), general framework

## Content

The shape of the resource gradient — how resources are distributed across terrain — determines what kind of civilization develops. Three patterns from the same framework:

**Horizontal accessibility (Plains):** North-south seasonal migration provides continuous access to resources without storage. No pressure to settle. No sedentary complexity. Mobile optimization. (Ref 010)

**Vertical density (Inca/Andes):** Altitude stacking puts multiple ecological zones within a day's walk. Sedentary, but compact — no pressure to expand laterally because everything you need is available vertically. Denser population is viable because:
- Food diversity without geographic spread
- Reduced sanitation problems from not needing to sprawl
- Not every person needs a large plot — the vertical supermarket feeds a concentrated population

**Linear distribution (River valleys):** Resources spread downstream along the river. Sedentary and spread out. Pressure to trade along the line, administer the line, defend the line. This produces the bureaucratic state almost automatically.

**The computer science analogy:** This is vertical scaling vs. horizontal scaling. The Andes scaled vertically — more resources per unit of horizontal space. The Plains scaled horizontally — more space accessed per unit of time via migration. River valleys scaled linearly — resources distributed along one axis requiring administration. Each scaling model produces a different civilization type with different complexity characteristics.

**The expansion pressure test:** Vertical resource density (Inca) reduces the incentive to expand and trade. You have what you need locally. Linear resource distribution (Nile, Tigris) creates trade pressure because no single point on the line has everything. Horizontal accessibility (Plains) eliminates the need for both trade and storage. The *shape* of the gradient determines whether you get isolationist, trade-networked, or mobile cultures.

## Why this matters

This is a more precise formulation of the Chapter 1 framework than "geography sets the distribution." The specific claim: **the shape of the resource gradient determines the type of civilization.** Not just whether civilization develops, but *what kind* — mobile, compact-sedentary, or expansionist-sedentary. 

This also connects the Plains argument (ref 010), the river spine argument (Ch. 2), and the Inca analog argument (ref 017) into a single unified model. They're not three separate observations — they're three outputs of the same function with different inputs.

And it addresses the "but what about..." objections preemptively: the framework doesn't predict the same civilization everywhere. It predicts *different* civilizations from *different* gradients, and the specific differences are predictable from the terrain.
