# Interactive Maps — Lessons from Prior Dissertation and High-Value Map Moments

**Source:** Beta seminar discussion, 2026-04-14; Dave's prior work
**Type:** reference
**Relevant to:** All chapters — visual presentation layer

## Content

Dave's prior dissertation (eye-tracking research, defended 2013) was web-based with interactive visualizations: https://github.com/gotoplanb/discriminating-news-reading-behavior — JS/CSS/Ruby stack. Lesson learned: scope the interactives tightly to the argument.

**The concept:** Interactive maps with a timeline stepper showing borders changing over a persistent terrain base layer. The contrast between stable topography and shifting political borders does the argument visually — the terrain stays constant while the borders thrash around it. That IS the thesis, rendered as an image.

**Tech leads (from Beta):** Leaflet.js or MapLibre with timeline slider, GeoJSON boundary files per era, terrain as persistent base.

**High-value map moments identified so far:**
1. The Silk Road corridor with terrain showing why those specific routes — the funnels, fans, and knot visible in topography
2. The parallel conquest map — American West and Russian steppe simultaneously, same decade, split screen or overlay
3. Soviet border drawing in Central Asia — deliberate ethnic mixing visible as borders that ignore terrain/ethnic boundaries
4. The Aral Sea timeline — before/after satellite imagery. The canal that destroyed the lake. Viscerally devastating. Might be the single most powerful image in the book.
5. The Nile/Amazon/Volga triptych — same scale, different rivers, different outcomes visible in the terrain
6. The Inca mountain spine — altitudinal zones visible in terrain coloring, showing vertical resource density

**Editorial discipline:** Each map answers a specific analytical question, not just "here's how borders changed." The question each map answers should be statable in one sentence. If it can't, the map is decoration.

## Why this matters

The book is open-access and web-native. Interactive maps are not supplementary — they're a core part of the argument. The terrain base layer literally shows the reader what the book is about: the geography doesn't move. Everything else does.
