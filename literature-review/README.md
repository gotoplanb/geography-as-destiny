# Literature Review

One file per source. Each file follows the same template:

- **Bibliographic info** — author, title, year, publisher
- **What it argues** — the source's own thesis in a few sentences
- **What we use it for** — how it serves our argument (which lens, which chapter)
- **Key excerpts and references** — specific passages, page numbers, data points, with annotation
- **Limitations** — where the source is weak, what it doesn't cover, what we disagree with
- **Connections** — how it relates to other sources in this review

Files are named `lastname-short-title.md` (e.g., `morrison-russian-conquest.md`).

Sources accumulate over time. A file can start sparse and get annotated as Dave reads deeper.
